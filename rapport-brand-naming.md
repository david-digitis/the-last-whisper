## Resume

"The Last Whisper" etait le nom placeholder initial. Ce rapport a analyse 20 candidats et recommande **Dikto** (du latin *dictare*) -- le nom finalement retenu. L'application merite un nom qui communique instantanement sa triple valeur : **voix vers texte**, **intelligence**, **vitesse**. Les 20 candidats ont ete verifies sur GitHub et npm, classes par famille semantique, avec un Top 3 argumente.

---

## Profil de la marque

### Ce que le nom doit evoquer
- **Voix** : l'input fondamental est la parole humaine
- **Vitesse** : 50ms-2s de latence, push-to-talk, tout est instantane
- **Transformation** : la voix devient texte, email, traduction
- **Controle local** : privacy, offline, pas de cloud pour le STT
- **Outil de power user** : pas un gadget, un compagnon quotidien

### Contraintes de positionnement
- Doit fonctionner comme commande CLI (`dikto record`, `spokn --help`)
- Prononçable sans ambiguite en francais ET en anglais
- Pas de collision avec un projet GitHub connu (190+ stars = elimine)
- Disponible sur npm (verifie le 2026-03-25)

### Concurrents nommes a eviter de copier
- Whisper (OpenAI) -- trop proche du nom actuel
- Dictanote, Otter, Descript -- generiques
- DeepL -- deja installe dans les esprits pour la traduction

## Top 3 Recommandes

### 1er choix : DIKTO

**Pourquoi c'est le meilleur nom.**

- **Etymologie forte** : du latin *dictare* -- le mot qui a donne "dicter", "dictaphone", "dictee" dans toutes les langues romanes. Tout le monde comprend instantanement.
- **Prononciable partout** : "DIKT-O" se dit de la meme façon en francais, anglais, allemand, espagnol. Pas d'ambiguite.
- **CLI parfait** : `dikto`, `dikto record`, `dikto translate`. Court, tape bien, pas de conflit avec des commandes existantes.
- **Memorisable** : 5 lettres, 2 syllabes. Comparable a Figma, Slack, Notion en termes de compacite.
- **Extensible** : "Dikto AI", "Dikto Pro", "Dikto Cloud" fonctionnent naturellement si le projet grandit.
- **Differenciateur** : ne ressemble a aucun concurrent (Whisper, Otter, Descript). Sonne technique sans etre froid.
- **Pitch en une phrase** : "Dikto -- la voix qui ecrit."

**Risques** : Aucun projet significatif sur GitHub ou npm. Le domaine dikto.io/dikto.app devrait etre verifie.

---

### 2eme choix : SPOKN

**Le choix tech/minimaliste.**

- **Visuel frappant** : le 'e' manquant de "spoken" cree une tension visuelle memorable (comme Flickr, Tumblr a l'epoque).
- **Signification immediate** : "spoken" = "prononce". Pas besoin d'explication.
- **CLI excellent** : `spokn`, `spokn --paste`, `spokn clip`. Compact, rapide a taper.
- **Tonalite** : sonne dev-friendly, open source, pas corporate. Parfait pour GitHub.
- **Differenciation** : aucun outil connu ne porte ce nom.

**Risques** : La prononciation peut varier (SPOK-N ou SPOKEN?). Moins intuitif pour un francophone pur.

---

### 3eme choix : VOXPASTE

**Le choix descriptif qui ne ment pas.**

- **Auto-explicatif** : "Vox" (voix) + "Paste" (coller). Le workflow entier dans le nom.
- **SEO naturel** : quand quelqu'un cherche "voice paste tool", VoxPaste apparait.
- **Mecanisme clair** : tu parles, ça colle. Zero marketing necessaire pour faire comprendre le produit.
- **CLI** : `voxpaste` est un peu long mais fonctionne. Alias `vp` possible.

**Risques** : Deux mots colles = moins "premium" qu'un mot invente. Ne capture pas les features IA/traduction (seulement le push-to-talk). Pourrait sembler reducteur si le produit evolue au-dela du paste.

## Actions recommandees

### 1. Immediate : Valider le nom final

- Choisir entre Dikto, Spokn ou VoxPaste (ou un autre de la liste)
- Verifier la disponibilite du domaine (dikto.dev, dikto.app, dikto.io)
- Creer le repo GitHub sous le nom definitif
- Reserver le package npm avec un `package.json` minimal

### 2. Court terme : Migration de marque

- Renommer le repo GitHub `the-last-whisper` -> `dikto` (fait)
- Mettre a jour `package.json` (name, description)
- Mettre a jour le README, le titre de la fenetre Electron, le tray tooltip
- Creer un logo minimal (icone + wordmark)
- Definir la tagline officielle (ex: "Dikto -- la voix qui ecrit" / "Dikto -- voice to text, instantly")

### 3. Long terme : Construction de marque

- Developper un design system coherent (couleurs, typographie, icones)
- Creer une landing page sous le domaine reserve
- Etablir les brand guidelines pour les contributions (screenshots, badges)
- Considerer le depot de marque si le projet gagne en traction
